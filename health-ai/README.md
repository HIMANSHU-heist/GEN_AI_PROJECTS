# 🧠 Health AI – Food & Medicine Image Analyzer

**Health AI** is a Generative AI powered web application that analyzes **food and medicine images** and provides useful health insights.

Users can upload an image and the system will generate information like **nutrition, calories, medicine usage, side effects, and precautions.**

---

## 🚀 Features

* 🍎 Food image analysis
* 💊 Medicine identification
* 📊 AI generated health insights
* 🖼 Drag & drop image upload
* ⚡ Fast API response
* 📱 Clean web interface

---

## 🛠 Tech Stack

**Frontend**

* HTML

**Backend**

## ⚙ Setup

Install dependencies

pip install -r requirements.txt

Create `.env` file

GROQ_API_KEY=your_api_key_here

Run backend

python app.py

Open frontend

frontend/index.html
