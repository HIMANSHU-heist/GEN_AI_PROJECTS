# 🚀 HireSense AI

### AI-Powered Candidate Ranking System

---

## 🧠 Problem

Hiring is broken.

- Too many resumes  
- Keyword-based filtering fails  
- Strong candidates get missed  
- No real insight into skills or impact  

---

## 💡 Solution

**HireSense AI** acts like an intelligent hiring panel.

- Analyzes resumes deeply (not just keywords)  
- Evaluates real skills, projects, and impact  
- Ranks candidates from best to worst  
- Returns only **top N candidates** you need  

---

## 🔥 Features

- 🧠 Smart Resume Analysis  
- 📊 AI-Based Scoring (Match, Impact, Depth)  
- 🏆 Top Candidate Selection  
- 📌 Detailed “Why Best” Explanation  
- ⚖️ FAANG-Level Evaluation Logic  

---

## ⚡ How It Works

1. Upload resumes  
2. Enter role, skills, seniority  
3. Choose number of candidates  
4. Get **top ranked candidates with insights**  

---

## 🛠 Tech Stack

- Backend: FastAPI, Python  
- AI: Groq (LLM)  
- Frontend: HTML, CSS, JavaScript
